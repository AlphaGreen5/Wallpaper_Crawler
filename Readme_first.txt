How to install:

1. open CMD prompt or Shell.
2. write bellow commands one by one
   pip install bs4
   pip install xlib
   pip install requests



To execute in linux OS:

Open the Wallpaper_Crawler.py file in a Text editor
and then paste the below command at the top and then save.

#! /usr/bin/python3

Then open the shell and go to the Wallpaper_Crawler directory in shell
and execute the below command

python Wallpaper_Crawler.py
